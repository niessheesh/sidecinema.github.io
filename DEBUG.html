  4. Lihat apakah ada pesan seperti:

Jika Tidak Ada Pesan Console

Kemungkinan masalah:

  • ‚ùå script.js tidak ter-load (periksa di tab Network)
  • ‚ùå Ada error syntax (periksa di tab Console)

Solusi:
